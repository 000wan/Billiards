# Billiards
Four-ball Billiards Pygame in 2020

Recently updated: 2022/7/25

![capture_gif](https://user-images.githubusercontent.com/87213416/187011894-3ff865c1-691f-45b9-b417-436bc510dade.gif)

## Required Python Modules
    pygame
    pygame_gui
    numpy

Download modules with 'pip install ~'
